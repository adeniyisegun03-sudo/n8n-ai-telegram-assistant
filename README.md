# n8n-ai-telegram-assistant
AI-powered Telegram assistant built with n8n
## AI-Powered Telegram Assistant (n8n)

This repository contains an n8n workflow that automates Telegram interactions using an AI agent.

### Features
- Telegram trigger for incoming messages
- AI agent powered by OpenAI
- Memory for conversational context
- Reads data from Google Sheets
- Creates events in Google Calendar
- Sends messages via Gmail and Telegram

### Tools Used
- n8n
- OpenAI API
- Telegram Bot API
- Google Sheets
- Google Calendar
- Gmail

### Purpose
This project demonstrates how AI and automation can be used to streamline communication, scheduling, and notifications.
